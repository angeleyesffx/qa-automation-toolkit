# Portfolio

This folder contains a self-contained portfolio website for **Priscilla R. Martins**.

## Files

- `/tmp/workspace/angeleyesffx/lazy_qa/portfolio/index.html` - main page
- `/tmp/workspace/angeleyesffx/lazy_qa/portfolio/css/style.css` - styles and responsive layout
- `/tmp/workspace/angeleyesffx/lazy_qa/portfolio/js/main.js` - theme toggle, smooth scroll, mobile navigation, and reveal effects

## Open locally

You can open the portfolio in either of these ways:

1. Open `/tmp/workspace/angeleyesffx/lazy_qa/portfolio/index.html` directly in a browser.
2. Serve the repository locally and open it in a browser:

   ```bash
   cd /tmp/workspace/angeleyesffx/lazy_qa
   python -m http.server 8000
   ```

   Then visit `http://127.0.0.1:8000/portfolio/`.

## Features

- Responsive layout with mobile navigation
- Light and dark theme toggle with saved preference
- Smooth scrolling navigation
- Vertical experience timeline
- Skills badges
- Structured placeholder sections for education, certifications, languages, and courses

## Notes

- The website uses only local HTML, CSS, and JavaScript files.
- No external CDN assets or third-party front-end dependencies are required.
